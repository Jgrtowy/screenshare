# Discord screenshare... a bit improved.

99.9% vibecoded (well, I had to fix some things myself)(too lazy to actually code).

Using [SRS](https://ossrs.net/lts/en-us/) as a *bridge* between OBS stream and the front-end Next.js app. Targeted to use for movie nights without depending on in-app screenshare. Provides better looking picture as well as higher framerates so you can actually watch the movie, not a slideshow with countable pixels.

For now it works, needs polishing tho, might do it later.
